# Exercise 2 - E-commerce Platform Search Function

## Objective

The objective of this exercise is to implement Linear Search and Binary Search algorithms and compare their performance in searching products on an e-commerce platform.

## Description

In an e-commerce platform, users search for products frequently. To provide search results quickly, efficient searching algorithms are required. In this exercise, Linear Search and Binary Search are implemented to find products based on their names.

## Product Details

Each product contains the following attributes:

* Product ID
* Product Name
* Category

## Linear Search

Linear Search checks each product one by one until the required product is found.

### Time Complexity

* Best Case: O(1)
* Average Case: O(n)
* Worst Case: O(n)

## Binary Search

Binary Search works on sorted data and repeatedly divides the search space into two halves.

### Time Complexity

* Best Case: O(1)
* Average Case: O(log n)
* Worst Case: O(log n)

## Comparison

Binary Search is faster than Linear Search for large datasets because it reduces the number of comparisons required to find a product.

## Result

The Linear Search and Binary Search algorithms were implemented successfully and tested using sample product data. The search results were displayed correctly.
